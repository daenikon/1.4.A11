# pkgprobe-dk
- custom script to display useful info about the package:
    1.  maintainer(s)
    2.  description
    3.  upstream info
    4.  USE Flags
    5.  size

* `..-dk` is there to identify custom scripts
* location: `~/bash_scripts/pkgprobe-dk`
